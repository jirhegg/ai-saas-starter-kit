# 제품 개요

## 프로젝트 설명
AI 기반 코딩 보조 SaaS 플랫폼으로, 자연어 입력을 받아 프로덕션 레벨의 코드를 생성합니다.

## 핵심 기능
1. **자연어 → 아키텍처**: 간단한 설명으로 전체 시스템 설계
2. **스펙 생성**: 자동으로 상세한 기술 스펙 문서 작성
3. **코드 생성**: 테스트 포함 프로덕션 코드 자동 생성
4. **RAG 검색**: 벡터 기반 컨텍스트 검색 및 추천

## 기술 스택
- **프레임워크**: Next.js 16 (App Router)
- **데이터베이스**: Supabase (PostgreSQL + pgvector)
- **ORM**: Drizzle ORM
- **UI**: shadcn/ui + Tailwind CSS
- **상태 관리**: TanStack Query
- **인증**: Supabase Auth
- **배포**: Vercel

## 타겟 사용자
- 풀스택 개발자
- 스타트업 창업자
- 프로토타입 제작자
- 개발 팀 리더

## 비즈니스 목표
- 개발 속도 10배 향상
- 코드 품질 일관성 유지
- 보일러플레이트 자동화
- 베스트 프랙티스 자동 적용
